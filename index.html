My Projects

Booking Application

This is my first project for a company where the requirement was a full stack booking website. Using this website/application, users can book the venue on any specified date, and can make payment as well.

Details of this application:

  • If a booking had already been done, another user cannot book on the same date
  • A booking cannot be done on the same day, previous day/month/year
  • Payment Options - Account Transfer / Cash (By visiting the venue)
  • Upon succesfull completion of booking, the user gets an email confirmation and a booking receipt
  • The backend system verifies all the user provided details and validates for errors (if any).
  • The admin can view the list of bookings in the dashboard and can approve them after payment confirmation, once confirmed the user gets an confirmation email as well.

Class Representative Voting Application

A voting system where each Student can vote their Class Representatives(Candidates), this application has different login application & dashboard for student, candidate, and Admin. Candidates has own privileges to update their profile. Admin can add Candidates, Students from various sections and can group them based on section. A Student can only login and vote their candidate from section they belong to.
Application was designed keeping few things in mind

  • Each student can vote only once.
  • Student from section A cannot vote candidates from section B
  • The final result will be published by Admin and every candidates, students will get a mail for the same
This project is not hosted online, for more information please feel free to contact me.
